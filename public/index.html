🔧 Configuración: Communication Arrangement

Para usar la funcionalidad de Application Jobs, necesitas configurar el escenario SAP_COM_0326 en tu sistema SAP IBP:

  1. Communication User: Crea un usuario técnico en Maintain Communication Users
  2. Communication System: Crea un sistema en Communication Systems apuntando a tu tenant IBP
  3. Communication Arrangement: Crea un arrangement con escenario SAP_COM_0326 (External Scheduler - Application Job Administration Integration)
  4. Asigna el Communication User y System al arrangement
  5. Configura autenticación Basic Auth (usuario y contraseña)
  6. Usa las mismas credenciales en el panel de conexión de GoSCM

Este Communication Arrangement expone el servicio BC_EXT_APPJOB_MANAGEMENT que permite leer los Application Jobs y sus pasos.
