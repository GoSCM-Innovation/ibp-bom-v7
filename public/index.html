Conexión a SAP IBP
URL base requerida
https://{instancia}-api.scmibp.ondemand.com
Reemplaza {instancia} con el ID de tu tenant SAP IBP.
Requisitos de conexión
✓  Protocolo HTTPS obligatorio
✓  Puerto 443
✓  Autenticación HTTP Basic Auth (usuario + contraseña)
✓  Planning Area configurada en SAP IBP
✓  Versión (opcional — vacío = Baseline)
Tipo de usuario requerido
✓  Debe ser un Communication User (no usuario de negocio)
✓  Requiere rol de administrador en SAP IBP
✓  La contraseña no debe estar vencida
✓  El usuario debe estar asignado al Communication Arrangement
Cómo crearlo
1.  Ir a SAP IBP → Administration
2.  Communication Users → New
3.  Tipo: Communication User
4.  Asignar contraseña segura
5.  Asignar rol: IBP Administrator
Escenario requerido
SAP_COM_0720
Este escenario activa automáticamente las dos APIs que usa esta web:
✓  /IBP/MASTER_DATA_API_SRV
✓  /IBP/PLANNING_DATA_API_SRV
Pasos de configuración
1.  Crear Communication System (inbound)
2.  Asignar el Communication User
3.  Crear Communication Arrangement
4.  Seleccionar escenario SAP_COM_0720
5.  Anotar la URL de integración generada

¿Por qué se usa un proxy?
Los navegadores bloquean llamadas directas a dominios externos por política CORS. Esta web usa un servidor proxy intermedio (/api/proxy) que reenvía las solicitudes a SAP IBP desde el servidor, evitando el bloqueo del navegador.
Navegador → /api/proxy → SAP IBP

🔧 Configuración: Communication Arrangement

Para usar la funcionalidad de Application Jobs, necesitas configurar el escenario SAP_COM_0326 en tu sistema SAP IBP:

  1. Communication User: Crea un usuario técnico en Maintain Communication Users
  2. Communication System: Crea un sistema en Communication Systems apuntando a tu tenant IBP
  3. Communication Arrangement: Crea un arrangement con escenario SAP_COM_0326 (External Scheduler - Application Job Administration Integration)
  4. Asigna el Communication User y System al arrangement
  5. Configura autenticación Basic Auth (usuario y contraseña)
  6. Usa las mismas credenciales en el panel de conexión de GoSCM

Este Communication Arrangement expone el servicio BC_EXT_APPJOB_MANAGEMENT que permite leer los Application Jobs y sus pasos.
